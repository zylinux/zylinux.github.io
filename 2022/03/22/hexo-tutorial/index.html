hexo_tutorial

Hexo
2022-03-22

1-node.js download and install
https://nodejs.org/en/download/
node-v16.14.2-x64.msi

2-git bash download and install
https://git-scm.com/downloads
Git-2.35.1.2-64-bit.exe

3-install hexo-cli
npm install -g hexo-cli

4-create a folder with name blog which contains your blog website
hexo init blog

5-go into your blog
cd blog
hexo g
hexo s

6-test your website locally
open your brower,put in the following address, you should be able to see webiste which you have created
localhost:4000

7-3 type of pages
post page:
hexo new a
draft page:
hexo new draft b
hexo server –draft
hexo publish b
normal page(like about page):
hexo new page c

8-front matter
$ cat source/_posts/min-test-doc1.md

title: min_test_doc1
date: 2022-03-22 10:06:36
tags: [Tag1, Tag2, Tag3]

9-scaffolds,each of md file under scaffolds controls each of file under source/
page.md controls every page , here are a b c.
post.md controls _posts
draft.md controls _draft

$ ls scaffolds/
draft.md page.md post.md

$ ls source/
_posts _draft a b c

one tip, if you want use different themes to new page, add layout: xxxx into scaffolds/xxx.md

10-tags & categories
just add tag to your md file

title: min_test_doc1
date: 2022-03-22 10:06:36
—tags: [Tag1, Tag2, Tag3]

just add categories to your md file

title: min_test_doc1
date: 2022-03-22 10:06:36
tags: [Tag1, Tag2, Tag3]
categories:

  • [Cat1, Cat1.1]
  • [Cat2]
  • [Cat3]
